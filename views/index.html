Currently, we're working on this app and it's in alpha. This page is going to serve as the state of the app until we're in a more usable state. Here's what works, what doesn't, and what's on the roadmap.

What Works

  • Create Curriculum
  • See All Curriculum
  • Create Assignments
  • See All Assignments
  • Create News
  • View News on the Today thing
  • Post "Community" Link (ie share something cool you found / learned from)
  • Post "Publisher" Link (ie share something you wrote and solicit feedback)
  • View "Community" vs "Publisher" links along with a "Newest"(mixed feed) of them.
  • Comment on an individual link (click on the discussion thing)

What's Next

  • New interface for planning individual curriculum (sketch)
  • Distinctify Curriculum from Units (ie, Functions from Models in IA-parlance)
  • Allow scheduling of Units
  • Allow instructors to assign assignments (or auto-assign assignments) (or allow students to declare they have started on an assignment if it's like, a choose-your-own adventure thing)
  • Add more meaningful onboarding, integrate with actual onboarding insights so we have that stuff written down and have more in software about the tn+1 / tn+2 student profiles
  • Add resources model, and have a listing of resources that are related to different entities (functions, models, assignments, discussions, "playlists", etc)
  • Bug reporting form
  • Feature request form
  • Skills Profiles
