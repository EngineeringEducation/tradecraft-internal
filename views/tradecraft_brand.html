Logo

Usage guidelines

There are 3 rules of logo usage:

  1. Don't ever stretch the logo.
  2. Leave 15% whitespace all the way around.
  3. If you're using a raster version, only make it smaller — never bigger (or pixelation will occur and you'll make children cry). Use vector versions when you can to be safe.

Download options

Use the most appropriate logo for the job. When in doubt, refer to rule #3 above.

Typography

Use no more than 2 typefaces in any given piece of collateral (print or web).

Headline copy

GOTHAM LIGHT. All caps, with line-height / leading that is 200% larger than the type size. For instance, if you're setting a headline at 48px, the leading should be 96px.

Download Gotham *
* Don't share outside TC

Body copy

Sentinel book. Normal caps, and the line-height is 1.5x the point size. For instance, the default size on the website is 18px large, with a line height of 27px.

Download Sentinel *
* Don't share outside TC
